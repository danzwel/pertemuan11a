# Notes Hybrid App (Flutter + Firebase)

Aplikasi **Notes Hybrid** adalah aplikasi pencatatan sederhana berbasis **Flutter** yang terintegrasi dengan **Firebase Authentication**, **Firebase Analytics**, serta mendukung **notifikasi lokal**.  
Aplikasi ini dibuat untuk memenuhi kebutuhan **praktikum Pemrograman Mobile**.
